# Employee Management System

A Spring Boot application built to demonstrate Spring Data JPA, Hibernate, and relational database concepts through a real-world Employee Management System.

## Tech Stack

- Java 21
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Maven
- Lombok
- Bean Validation

## Project Goals

This project is designed to cover:

- CRUD Operations
- One-To-One Relationships
- One-To-Many Relationships
- Many-To-One Relationships
- Many-To-Many Relationships
- Query Methods
- JPQL
- Native SQL Queries
- Pagination
- Sorting
- Specifications
- DTO Pattern
- Exception Handling
- Auditing
